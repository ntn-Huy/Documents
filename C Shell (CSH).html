C Shell từ cơ bản đến nâng cao

Tác giả: Nguyễn Thành Nhật Huy

1. Tại sao học C Shell (csh)?

C Shell mang cú pháp của ngôn ngữ C vào môi trường Shell, cực kỳ phổ biến trong các công cụ thiết kế vi mạch (EDA Tools).

2. Hello World - Script đầu tiên

#!/bin/csh -f
echo "Hello, Computer Engineering!"

3. Biến (set) & Biến môi trường (setenv)

set project = "Lab_01"
setenv TOOLS_PATH "/usr/local/bin"

4. Toán tử Số học & Bitwise

Csh cho phép dùng toán tử bitwise (AND, OR, Shift) trực tiếp.

@ a = 5
@ b = ( $a << 2 ) # Dịch trái 2 bit

5. Các cách thực hiện phép toán (+ - * /)

Tùy vào nhu cầu tính số nguyên hay số thực, bạn có 3 cách viết chính.

Cách 1: Sử dụng '@' (Số nguyên - Khuyên dùng)

@ x = $a + $b
# Bắt buộc có khoảng trắng quanh dấu = và toán tử.

Cách 2: Sử dụng 'expr' (Cổ điển)

set mul = `expr $a \* $b` # Phải có dấu \ trước dấu *

Cách 3: Sử dụng 'bc' (Số thập phân)

set div = `echo "scale=2; $a / $b" | bc`

6. Biểu diễn số thập phân (Floating Point)

Do lệnh @ chỉ hỗ trợ số nguyên, ta dùng bc hoặc awk để xử lý số thực.

set res = `awk "BEGIN {print 10 / 3}"`
printf "Kết quả: %.2f\n" $res

7. Nhập xuất dữ liệu ($<)

set part = $<

8. Cấu trúc điều khiển If Else

if ( $a > 10 ) then ... endif

9. Cấu trúc Switch Case

switch ( $argv[1] ) ... endsw

10. Câu lệnh Goto & Nhãn (Label)

11. Vòng lặp Foreach

foreach i ( 1 2 3 ) ... end

12. Vòng lặp While

13. Break & Continue

15. Xử lý Chuỗi ký tự
