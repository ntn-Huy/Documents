Học C Shell (csh) - Từ Cơ bản đến Nâng cao

Tác giả: Nguyễn Thành Nhật Huy

1. Tại sao học C Shell (csh)?

C Shell được tạo ra bởi Bill Joy tại Berkeley, mục tiêu là mang cú pháp của ngôn ngữ lập trình C vào môi trường Shell. Nó cực kỳ phổ biến trong các hệ thống Unix đời đầu và hiện nay vẫn là tiêu chuẩn trong các công cụ thiết kế vi mạch (EDA Tools).

  • Cú pháp điều khiển (If, Switch) giống hệt C.
  • Cơ chế Alias mạnh mẽ.
  • Phù hợp cho các kỹ sư làm việc với hệ thống phần cứng.

2. Hello World - Script đầu tiên

Mọi script chuyên nghiệp đều bắt đầu bằng #! (Shebang) và tùy chọn -f để bỏ qua file cấu hình cá nhân, giúp script chạy ổn định hơn.

#!/bin/csh -f
echo "Hello, Computer Engineering!"
echo "Kernel version: `uname -r`"

3. Biến (set) & Biến môi trường (setenv)

Trong Csh, set dùng cho biến cục bộ, còn setenv dùng cho biến toàn cục của hệ thống.

# Biến cục bộ
set project = "Lab_01"
echo "Working on $project"

# Biến môi trường (Không dùng dấu =)
setenv TOOLS_PATH "/usr/local/bin"

4. Toán tử Số học & Bitwise

Đây là điểm mạnh của Csh. Bạn có thể dùng các phép toán Bitwise (dịch bit, AND, OR) trực tiếp như trong C.

@ a = 5
@ b = ( $a << 2 ) # Dịch trái 2 bit (5 * 4 = 20)
@ c = ( $b & 15 ) # Phép AND logic bit
echo "Kết quả: $c"

5. Nhập xuất dữ liệu ($<)

Để nhận dữ liệu từ người dùng, C Shell sử dụng ký hiệu đặc biệt $<.

echo -n "Nhập tên linh kiện: "
set part = $<
echo "Bạn đã nhập: $part"

6. Cấu trúc điều khiển If Else

if ( $a > 10 ) then
    echo "Lớn hơn 10"
else
    echo "Nhỏ hơn hoặc bằng 10"
endif

7. Cấu trúc Switch Case

Dùng để thay thế chuỗi If-Else dài, rất hiệu quả khi kiểm tra tham số dòng lệnh.

switch ( $argv[1] )
    case "run":
        echo "Đang khởi chạy..."
        breaksw
    case "stop":
        echo "Đang dừng..."
        breaksw
    default:
        echo "Lệnh không hợp lệ"
        breaksw
endsw

8. Câu lệnh Goto & Nhãn (Label)

Giống như ngôn ngữ C hoặc Assembly, goto giúp nhảy đến một vị trí được đánh dấu bởi nhãn.

goto START_JOB

SKIP_SECTION:
echo "Phần này bị bỏ qua"
exit

START_JOB:
echo "Bắt đầu công việc tại đây"

9. Vòng lặp Foreach

Dùng để duyệt qua một danh sách các phần tử hoặc các file.

foreach i ( 1 2 3 4 5 )
    echo "Số: $i"
end

10. Vòng lặp While

@ n = 1
while ( $n <= 3 )
    echo "Lặp lần: $n"
    @ n++
end

11. Break & Continue

break dùng để thoát vòng lặp lập tức, continue dùng để nhảy sang lần lặp kế tiếp.

foreach num ( 1 2 3 4 5 )
    if ( $num == 3 ) continue
    if ( $num == 5 ) break
    echo "Value: $num"
end

12. Mảng 1 Chiều (Index từ 1)

Lưu ý: $array[0] trong C tương đương $array[1] trong Csh.

set my_list = ( "A" "B" "C" )
echo "Phần tử đầu: $my_list[1]"
echo "Số lượng: $#my_list"

13. Xử lý Chuỗi ký tự

Csh cho phép nối chuỗi đơn giản bằng cách viết chúng cạnh nhau.

set str1 = "Computer"
set str2 = "Engineering"
set full = "${str1}_${str2}"
echo "$full" # Output: Computer_Engineering

14. Đọc Ghi File & Redirection

Sử dụng >& để chuyển hướng cả đầu ra (stdout) và lỗi (stderr) vào file.

echo "Dữ liệu mẫu" > data.txt
gcc main.c >& compile.log # Ghi log cả lỗi biên dịch

15. Quản lý Tiến trình (Jobs Control)

Đây là kỹ năng nâng cao để quản lý nhiều tác vụ chạy song song.

sleep 100 & # Chạy ngầm (Background)
jobs # Xem danh sách tiến trình
fg %1 # Đưa tiến trình 1 lên phía trước (Foreground)
stop %1 # Tạm dừng tiến trình
